macOS setup:

  1. Open System Settings
  2. Go to Accessibility → Spoken Content
  3. Click System Voice → Manage Voices...
  4. Search "Thai" → Download Kanya
  5. Restart Chrome after download

Windows setup:

  1. Open Settings → Time & Language
  2. Click Language & region → Add a language
  3. Search "Thai" and install it
  4. Make sure Text-to-speech is checked
  5. Restart Chrome after install

Linux setup:

  1. Install espeak-ng: sudo apt install espeak-ng
  2. Or use Chrome — it may include Google Thai voice online
  3. Restart Chrome after install

Highlight Thai text on any page, then:

  • Right-click → Read with Poodthai
  • Or press Alt + P
